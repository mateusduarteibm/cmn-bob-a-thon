## Sobre o Bob-a-thon Learning Platform

O Bob-a-thon é uma plataforma educacional de código aberto para aprendizado de desenvolvimento full-stack moderno. O projeto demonstra boas práticas de arquitetura, desenvolvimento e documentação de software.

### Arquitetura

- **Frontend (FED)**: Aplicação Angular standalone com micro-frontends
- **Backend (BFF)**: API Spring Boot seguindo padrões de Clean Architecture

### Objetivo

Este inventário serve como documentação técnica completa dos repositórios, facilitando:
- Onboarding de novos desenvolvedores
- Compreensão da arquitetura do sistema
- Manutenção e evolução do código
- Aprendizado de boas práticas de desenvolvimento
